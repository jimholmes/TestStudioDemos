Testing for today's date

This page and the accompanying test project show how to test for today's date. This page gets today's day, also tomorrow's, and renders them in the table cells below.

The accompanying Test Studio project has a simple coded step which grabs today's and tomorrow's date via DateTime.Now, then passes that as a data bound parameter to the four Verify steps in the test.

Please note you'll have to change the date format in your Test Studio test to match whatever format is rendered on your actual page.
